# Cyber Security Task 1

## Project Overview
This project is based on testing vulnerabilities in a web application using OWASP Juice Shop.

## Tools Used
- OWASP ZAP
- Node.js
- OWASP Juice Shop

## Vulnerabilities Found
- Cross Site Scripting (XSS)
- SQL Injection
- Broken Authentication
- Security Misconfiguration

## How to Run Project
1. Clone the repository
2. Run npm install
3. Run npm start
4. Open http://localhost:3000
